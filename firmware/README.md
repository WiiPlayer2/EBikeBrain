### Bafang Display Extender Cable Colors

| Bafang | Prototype | Function             |
|--------|-----------|----------------------|
| Red    | Black     | GND                  |
| Black  | Green     | TTL Motor -> Display |
| Blue   | Blue      | PL                   |
| Green  | Yellow    | TTL Display -> Motor |
| Yellow | Red       | P+                   |
